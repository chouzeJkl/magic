### 垂直居中
```
.vertical-center {
   position: relative;
   top: 50%;
   transform: translateY(-50%);
}
```
这种方式的垂直对齐主要应用在块状元素中，缺点是外部容器需要设置height，有兼容性的问题，但写了前缀绝大部分的浏览器都可以兼容，除了IE那几个。。。

对于单行文本来说
```
.span-center {
   height: 40px;
   line-height: 40px;
}
```
### 水平对齐

文本居中
```
.text {
   text-align: center;
}
```
定宽的块元素居中
```
.block {
  margin: 0 auto;
}
```
